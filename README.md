# 🎓 Student Management System

A simple and beginner-friendly **Student Management System** built using **Python**. This project demonstrates core Python concepts such as Object-Oriented Programming (OOP), JSON, File Handling, Exception Handling, and CRUD (Create, Read, Update, Delete) operations.

---

## Features

- ➕ Add Student
- 🔍 Search Student
- ✏️ Update Student
- 🗑️ Delete Student
- 👤 View Student Details
- 📋 View All Students
- 💾 Save Data Permanently using JSON
- 📂 Load Existing Student Data Automatically
- 🚪 Menu-Driven Console Interface

---

## ▶️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/nareshnandan/student-management-system.git
```

2. Open the project folder

```bash
cd student-management-system
```

3. Run the program

```bash
python main.py
```

## Data Storage

Student information is stored permanently in the `students.json` file.

Example:

```json
{
    "101": {
        "name": "Naresh",
        "age": "24",
        "course": "Python"
    }
}
```
